# BiliArchive v0.1.0

BiliArchive 是一个面向 Windows 桌面场景的 Bilibili 内容归档工具。

首个可用版本已提供这些能力：

- 保存视频基础信息
- 抓取一级评论与子评论
- 抓取并导出字幕
- 导出 JSON 与 Markdown
- 支持使用 Cookie 或 SESSDATA 提高部分接口稳定性
- 提供 Windows 图形客户端与 EXE 打包产物

说明：

- 登录信息与本地输出目录只保存在本机
- `output/`、`dist/`、`.biliarchive.local.json` 默认不纳入版本控制
- 如果需要完整使用说明，请查看仓库文档
